<h1 align="center">🌟 About The Project</h1>

<table>
<tr>

<td width="60%">

<h3>🚀 Food Classification Using Deep Learning CNN</h3>

<p>
An advanced <b>Deep Learning & Computer Vision based Food Classification System</b> built using 
<b>Python</b>, <b>TensorFlow</b>, <b>Flask</b>, and <b>Redis Database</b>.
</p>

<p>
This AI-powered application can classify <b>34 different food categories</b> using multiple deep learning architectures.
</p>

<p>
The project uses:
</p>

<ul>
<li>🧠 Custom CNN Architecture</li>
<li>⚙️ VGG16 Transfer Learning</li>
<li>🚀 ResNet50 Transfer Learning</li>
<li>🔴 Redis Database Integration</li>
<li>🌐 Flask Web Application</li>
<li>📊 Real-Time AI Prediction</li>
</ul>

<p>
One of the most exciting parts of this project is the integration of <b>Redis Database</b>.
Food nutrition data is stored inside Redis and dynamically retrieved into the UI during prediction.
</p>

<p>
This project demonstrates how modern AI systems combine:
</p>

<ul>
<li>✅ Deep Learning</li>
<li>✅ Computer Vision</li>
<li>✅ Redis Integration</li>
<li>✅ Flask Deployment</li>
<li>✅ Real-Time Prediction</li>
<li>✅ Production-Level Workflow</li>
</ul>

</td>

<h1 align="center">🚀 Installation & Setup</h1>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/SachinMuskudi/Food_Classification_Using_DeepLearning_CNN.git
</pre>

<h3>2️⃣ Navigate To Project Folder</h3>

<pre>
cd Food_Classification_Using_DeepLearning_CNN
</pre>

<h3>3️⃣ Install Requirements</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>4️⃣ Start Redis Server</h3>

<pre>
redis-server
</pre>

<h3>5️⃣ Load Nutrition Data Into Redis</h3>

<pre>
python Set_data_to_redis.py
</pre>

<h3>6️⃣ Download Model Weights</h3>

<pre>
Download from GitHub Releases section
</pre>

<h3>7️⃣ Run Flask Application</h3>

<pre>
python app.py
</pre>
